# Demo Knowledge-Base Documents

These Markdown files are self-written sample policies for local RAG demos.

Recommended upload order:

1. `sample_company_policy.md`
2. `sample_after_sales_policy.md`
3. `sample_return_policy.md`
4. `sample_order_abnormal_handbook.md`

They are designed to support policy Q&A, after-sales reasoning, and the multi-step abnormal-order report demo.
