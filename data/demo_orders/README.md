# Demo Order CSV Dataset

This directory contains deterministic simulated e-commerce data for SQL Agent and multi-step report demos.

## Files

- `demo_customers.csv`: 60 demo customers across 10 states.
- `demo_products.csv`: 40 demo products across 10 categories.
- `demo_orders.csv`: 320 demo orders over a simulated 90-day period.
- `demo_order_items.csv`: 400 order items.
- `demo_reviews.csv`: 320 reviews, including low-score records.
- `demo_after_sales.csv`: 153 after-sales records.

## Demo Coverage

The dataset covers `canceled`, `unavailable`, `delivery_delay`, `review_score <= 2`, `refund_request`, `product_damage`, `wrong_item`, `payment_issue`, and `customer_complaint` signals.

All records are simulated and safe for public GitHub demonstration.
