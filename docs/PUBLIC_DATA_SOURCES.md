# Public Data Sources

These official public sources are suitable for demo knowledge-base ingestion. Do not commit private, paid, or credential-protected documents.

## Apple

- Apple Newsroom: https://www.apple.com/newsroom/
- Apple Investor Relations: https://investor.apple.com/
- Apple SEC filings: https://www.sec.gov/edgar/browse/?CIK=320193

## Microsoft

- Microsoft News: https://news.microsoft.com/
- Microsoft Investor Relations: https://www.microsoft.com/en-us/Investor
- Microsoft SEC filings: https://www.sec.gov/edgar/browse/?CIK=789019

## Amazon

- Amazon News: https://www.aboutamazon.com/news
- Amazon Investor Relations: https://ir.aboutamazon.com/
- Amazon SEC filings: https://www.sec.gov/edgar/browse/?CIK=1018724

## Usage Guidance

- Prefer official PDFs, press releases, annual reports, developer documentation and product documentation.
- Store only source URLs and demo excerpts unless the license permits redistribution.
- Keep source metadata such as URL, title, publisher and retrieval date in `metadata_json`.
